Instructions

Upload class list

  1. Click the Browse button

    • Navigate to a class list file on your hard disk
    • The only requirement is that student names are contained in a column called name.

  2. Two new elements appear below the upload button.

  3. Click in the empty select box

    • Click the names of students
    • You can toggle whether you select student that are to be included or excluded from the pairing function - i.e. students that are present or absent.
    • The names you select will appear on the right.
    • There is also a class size counter, useful fo confirming you have selected the correct number of students.

  4. By default, the selection process assumes you are choosing students that are present.

    • If you have chosen absent students, click the toggle present/absent button.
    • This will invert your selection, still displaying the student that are in the class.s

  5. Once you are happy with your selection, move onto the Pair tab.

Creating pair groups

  1. Use the numeric input to decide on the number of people per group
  2. Click Pair Up
  3. Click the button again to re-shuffle
  4. Press the save button to write to csv - you can save to a nw file if you like.
  5. Repeat, overwriting the same file as in the last step - the record of previous pairs should accumulate.

Expected data format

For a brand new class, a CSV containing only a name column is all that is necessary.
The name column should contain the names of every student taking the class.

Once students have been paired, they are recorded in subsequent weeks. When saving the data, the original class list should be overwritten, and used in all following weeks. The data can be saved in a OneDrive folder that is shared with all staff involved in delivering any live sessions.
